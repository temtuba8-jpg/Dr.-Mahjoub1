# Dr. Mahjoub Orthopedics Website

**Project Overview**  
This is a web application for Dr. Mahjoub, an orthopedic and joint replacement consultant. The site provides information about the clinic, allows patient inquiries, and integrates with donation and contact features.  

**Live Demo:** [https://dr-mahjoub1.onrender.com](https://dr-mahjoub1.onrender.com)  

---

## Features

- Clinic information pages
- Contact form
- Donation integration
- Responsive HTML templates
- Fast deployment with Flask and Gunicorn

---

## Technologies Used

- **Python 3.11.9**  
- **Flask 2.3.4** – Web framework  
- **Gunicorn 21.2.0** – WSGI HTTP server  
- **PyMongo 4.5.1** – MongoDB integration  
- **dnspython 3.8.1** – DNS handling for PyMongo
